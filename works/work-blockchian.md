## Hyper Ledger Fubricブロックチェーン事業

### 概要
ブロックチェーン技術を活用したiOSアプリ開発・リード・マネージを担当。iOSチームでエンジニアのタスクマネジメントを 行いつつ、Clean Architecutreやアジャイル開発を意識した開発を行なっています。モバイル・サーバー間の暗号化・ 電子署名などを利用した本人認証なども対応しました。また、CI構築などの開発環境構築なども行っていました。

モバイル開発メンバー6 (iOS 3/Android 3)人、開発メンバー13人程度、会社全体18人程度。

*言語*

- Swift / Java
- Golang

*アーキテクチャ・技術知識*

- VIPER Architecture
- Clean Architecture
- アプリからの秘密鍵の発行・署名

*環境*

- GitHub
- Docker
- Fabric
- gRPC

### 課題１
マネージャーが組織全体の一人のみで各開発マネージャーがおらず、課題の発見・解決やモバイル開発の技術リードができる人がいない。また、CIやディストリビューションができる環境が整っておらず、マネージャーから業務を移譲してもらい、モバイルチームの開発リード、Circle CIを使ったビルドの自動化・ディストリビューション環境の構築などを行いました。
